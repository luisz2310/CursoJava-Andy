## 🧪 Ejercicios incluidos

### 🔢 NumeroMayor
Compara dos números y muestra:
- cuál es mayor
- o si son iguales

💡 Aprendes:
- diferencia entre `if` y `else if`

---

### 🔁 NumerosImpares
Determina si un número es:
- PAR
- IMPAR

💡 Aprendes:
- operador `%` (módulo)

---

### 🔄 SumaPrimerosNnumeros
Imprime números de `0` a `N`

💡 Aprendes:
- construcción de un `for` paso a paso
