### Lógica de diagramas → Java
📁 Convertir un diagrama en código Java

- Equivalentes entre pasos de un diagrama de flujo y Java  
- Inicio → `main()`  
- Procesos → instrucciones secuenciales  
- Decisiones → `if / else`  
- Ciclos → `for / while`  
- Fin → final del programa
